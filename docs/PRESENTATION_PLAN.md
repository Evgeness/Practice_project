# План презентации на 5–7 слайдов

1. Проблема, цель и команда.
2. Архитектура: React, FastAPI, PostgreSQL, Elasticsearch, Redis.
3. Сценарий загрузки и индексации.
4. Сценарий поиска, русский анализатор и кеш.
5. DevOps: Docker Compose, CI, Prometheus/Grafana.
6. Тесты и метрики: coverage, E2E, нагрузка, Precision@3.
7. Демонстрация, ограничения и дальнейшее развитие.

Для демонстрации заранее очистить volumes, запустить систему, загрузить 2–3 небольших документа и подготовить три проверенных запроса.
