# Результаты тестирования

Эта папка предназначена для результатов, полученных на машине команды после запуска полного Docker-стека.

Перед сдачей рекомендуется добавить сюда:

- `load-test.html` и CSV-файлы Locust для 50 одновременных пользователей;
- `precision-at-3.json` с результатом 10 эталонных запросов;
- при необходимости скриншот покрытия pytest или HTML-отчёт покрытия.

Файлы результатов не генерируются заранее, потому что показатели зависят от компьютера, набора загруженных документов и текущего состояния Elasticsearch.
