````markdown
# University Knowledge Search

Интеллектуальная поисковая система по внутренней базе знаний университета.

---

## Требования

Перед запуском убедитесь, что установлены:

- Docker
- Docker Compose
- `make` (рекомендуется для Linux/macOS)

---

## Быстрый запуск

### 1. Подготовка конфигурации

Скопируйте шаблон переменных окружения:

```bash
cp .env.example .env
```
````

### 2. Запуск проекта

Соберите и запустите все сервисы в фоновом режиме:

```bash
docker compose up --build -d
```

После запуска приложение будет доступно по адресу:

```
http://localhost:3000
```

---

## Доступные сервисы

| Сервис      | Контейнер    | Адрес                      |
| ----------- | ------------ | -------------------------- |
| Frontend    | `front`      | http://localhost:3000      |
| Backend API | `app`        | http://localhost:8000      |
| Swagger UI  | `app`        | http://localhost:8000/docs |
| Prometheus  | `prometheus` | http://localhost:9090      |
| Grafana     | `grafana`    | http://localhost:3001      |

---

## Управление через Makefile

Для упрощения локальной разработки доступны следующие команды:

| Команда     | Описание                                                                         |
| ----------- | -------------------------------------------------------------------------------- |
| `make up`   | Сборка и запуск контейнеров Backend, Frontend, PostgreSQL, Elasticsearch и Redis |
| `make down` | Остановка и удаление контейнеров приложения                                      |
| `make lint` | Запуск линтеров и статических анализаторов кода                                  |

---

## Структура репозитория

```
.
├── backend/      # API (FastAPI), парсеры документов и бизнес-логика

├── monitoring/   # Конфигурации Prometheus и Grafana

```
